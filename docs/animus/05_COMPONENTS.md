# Additional Components

## SMCP - Sanctum Model Context Protocol

**Repository**: [smcp](https://github.com/AnimusUNO/smcp)

Internal MCP server for Animus that enables advanced agent tools and integrations:

- Provides access to external services (like CoinMarketCap for crypto data)
- Allows agents to extend capabilities through protocol-based tool integration
- Enables Model Context Protocol compliance for agent communication

SMCP serves as the bridge between Animus agents and external tools, services, and APIs. It implements the Model Context Protocol standard for seamless tool integration.

## Cochlea - Audio Processing

**Repository**: [cochlea](https://github.com/AnimusUNO/cochlea)

Audio Ingest System for Animus:

- Audio processing and voice interaction capabilities
- Processes voice input for agents
- Enables voice-enabled agents and audio-based interactions

Cochlea handles all audio-related processing, from speech recognition to audio generation, enabling agents to interact through voice interfaces.

## Animus Social

**Repository**: [animus-social](https://github.com/AnimusUNO/animus-social)

Specialized social media integration layer that enables agents like Bianca to engage authentically on platforms like Twitter, Bluesky, and Discord:

- Multi-platform support (X/Twitter, Bluesky, Discord)
- Persistent memory across platforms
- Queue management for robust notification processing
- Platform-specific tool registration
- Cross-platform identity consistency

## Modular Architecture & Privacy

Animus embraces a **modular, self-hosted design** that prioritizes privacy and autonomy:

- **Modular Design**: Deploy specialized agents such as deep research modules, sensory filters, and memory pruning tools with clean, plugin-driven integration
- **Infinite Context & Memory**: Advanced, layered memory management with automatic pruning, semantic enrichment, and archival storage
- **Robust Security & Privacy**: Secure, sandboxed execution and built-in access controls, ensuring data remains within your infrastructure without third-party exposure
- **Real-time Event Handling**: High-performance reflex processing through sensory middleware and proactive memory refinement
