# Animus Repositories

Animus is built as a modular ecosystem of specialized repositories, each handling specific cognitive functions.

## Repository Structure

### Core System Components
These are core components that flesh out the core systems required to run Animus agents.

#### installer
**Repository**: [AnimusUNO/installer](https://github.com/AnimusUNO/installer)
- Bootstrap installation and system setup
- Automated installation script
- Docker and Nginx configuration
- SSL certificate management

#### smcp
**Repository**: [AnimusUNO/smcp](https://github.com/AnimusUNO/smcp)
- Sanctum Model Context Protocol server for advanced agent tools
- Internal MCP server for Animus
- Tool integration and protocol compliance

### Research Repositories
These exist to help you build advanced functions, and the individual repos describe how these algorithms and methodologies fit into the larger framework.

#### thalamus
**Repository**: [AnimusUNO/thalamus](https://github.com/AnimusUNO/thalamus)
- Sensory routing and filtering system
- Advanced sensory processing
- Context management and real-time processing

#### cochlea
**Repository**: [AnimusUNO/cochlea](https://github.com/AnimusUNO/cochlea)
- Audio processing and voice interaction
- Voice-enabled agent capabilities
- Audio ingest system

### Integration Components
Utility repos we've created to demonstrate modalities or help you in your Animus journey.

#### animus-tui
**Repository**: [AnimusUNO/animus-tui](https://github.com/AnimusUNO/animus-tui)
- Terminal user interface for agent interaction
- Command-line interface for real-time agent chat
- Debugging and orchestration tools

#### animus-discord
**Repository**: [AnimusUNO/animus-discord](https://github.com/AnimusUNO/animus-discord)
- Discord integration and webhook processing
- Platform-specific agent deployment

#### animus-web-chat
**Repository**: [AnimusUNO/animus-web-chat](https://github.com/AnimusUNO/animus-web-chat)
- Flask-based web client for Animus
- Web-based agent interaction interface

#### animus-social
**Repository**: [AnimusUNO/animus-social](https://github.com/AnimusUNO/animus-social)
- Multi-platform social media integration
- Twitter/X, Bluesky, Discord support
- Queue management and persistent memory
